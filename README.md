## Linux cheat sheet

### Навигация и файлы

| КОМАНДА | ОПИСАНИЕ |
| --- | --- |
| `pwd` | показать текущую рабочую директорию |
| `cd /path` | `change directory` — перейти в директорию |
| `cd ~` | `change directory` — домашняя директория (~) |
| `cd ..` | `change directory` — на уровень выше (..) |
| `cd -` | `change directory` — предыдущая директория (-) |
| `ls` | список файлов в директории |
| `ls -l` | подробный вывод (права, дата, размер) |
| `ls -lh` | размер файлов в читаемом формате |
| `mkdir -p a/b` | создать директории (включая вложенные) |
| `touch file` | создать файл / обновить дату изменения |
| `cp src dst` | скопировать файл |
| `cp -r src dst` | скопировать директорию рекурсивно |
| `mv src dst` | переместить или переименовать |
| `rm file` | удалить файл |
| `rm -rf dir/` | удалить директорию рекурсивно |

### Просмотр файлов

| КОМАНДА | ОПИСАНИЕ |
| --- | --- |
| `cat file` | вывести всё содержимое файла |
| `head -n 5 file` | показать первые N строк |
| `tail -n 5 file` | показать последние N строк |
| `tail -f file` | следить за файлом в реальном времени |
| `less file` | постраничный просмотр (q — выход) |
| `grep "pat" file` | найти строки по шаблону |
| `grep -i "pat"` | поиск без учёта регистра |
| `grep -n "pat"` | показать номера совпавших строк |
| `grep -v "pat"` | показать строки БЕЗ шаблона |
| `grep -c "pat"` | подсчитать количество совпадений |
| `grep -r "pat".` | рекурсивный поиск по директории |
| `wc -l file` | подсчитать количество строк |
| `cmd > file` | перенаправить вывод в файл (перезапись) |
| `cmd >> file` | перенаправить вывод в файл (дозапись) |

### Права доступа

| **КОМАНДА** | **ОПИСАНИЕ** |
| --- | --- |
| `ls -l file` | просмотр прав доступа к файлу |
| `chmod +x file` | добавить право на исполнение |
| `chmod -w file` | убрать право на запись |
| `chmod 755 file` | `rwxr-xr-x` — скрипты, директории |
| `chmod 644 file` | `rw-r--r--` — обычный файл |
| `chmod 600 file` | `rw-------` — приватный файл |
| `chmod 777 file` | `rwxrwxrwx` — полные права для всех |
| `chown user file` | сменить владельца файла |
| `chown u:g file` | сменить владельца и группу |
| `umask` | маска прав доступа по умолчанию |

### Процессы

| **КОМАНДА** | **ОПИСАНИЕ** |
| --- | --- |
| `ps aux` | список всех процессов |
| `ps -ef` | полный список (unix-стиль) |
| `ps aux \| grep X` | найти процесс по имени |
| `top` | мониторинг ресурсов в реальном времени |
| `htop` | улучшенный интерактивный top |
| `kill PID` | отправить SIGTERM — просьба завершиться |
| `kill -9 PID` | SIGKILL — принудительное завершение |
| `kill -1 PID` | SIGHUP — перезагрузить конфигурацию |
| `killall имя` | завершить все процессы по имени |

### Сигналы

| **N** | **СИГНАЛ** | **ОПИСАНИЕ** |
| --- | --- | --- |
| 1 | SIGHUP | перезагрузить конфигурацию (nginx, sshd…) |
| 2 | SIGINT | прерывание (Ctrl+C) |
| 9 | SIGKILL | принудительное завершение — нельзя поймать |
| 15 | SIGTERM | корректное завершение (по умолчанию) |
| 19 | SIGSTOP | пауза процесса — нельзя поймать |
